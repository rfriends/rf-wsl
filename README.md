rfriends_wslはwindowsのwsl上にrfriends3をインストールするスクリプトです。  
   
cd ~/  
sudo apt-get install git -y  
rm -rf rfriends_wsl  
git clone https://github.com/rfriends/rfriends_wsl.git  
cd rfriends_wsl  
sh rfriends_wsl.sh
