rfriends_wslはwindowsのwsl上にrfriends3をインストールするスクリプトです。  
   
cd ~/  
wget -O rfriends_wsl-main.zip https://github.com/rfriends/rfriends_wsl/archive/master.zip  
unzip rfriends_wsl-main.zip  
cd rfriends_wsl-main  
sh rfriends3_wsl.sh
